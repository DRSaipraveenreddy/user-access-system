# User Access Management System

Full-stack role-based access system built with:

- **Node.js + Express**
- **TypeScript + TypeORM**
- **PostgreSQL**
- **JWT Authentication**

---

## Features

- User Signup & Login
- JWT-based authentication
- Role-based authorization (Employee / Manager / Admin)
- PostgreSQL + TypeORM integration
- RESTful APIs
